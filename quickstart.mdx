## Before you begin

<Checklist>
  <CheckItem>Node.js 18+ with npm or yarn</CheckItem>
  <CheckItem>Git access to your `mintlify-docs` repository</CheckItem>
  <CheckItem>Mintlify account with dashboard access</CheckItem>
  <CheckItem>OpenAI/Anthropic keys if you plan to use AI tooling during edits</CheckItem>
</Checklist>

## Step 1 · Clone & install

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone git@github.com:frinfo702/mintlify-docs.git
    cd mintlify-docs
    ```

    <Tip>Need to authenticate? Follow GitHub's <a href="https://docs.github.com/en/authentication/connecting-to-github-with-ssh">SSH setup guide</a>.</Tip>
  </Step>

  <Step title="Install the Mintlify CLI">
    ```bash
    npm i -g mint
    ```
    If you've installed it previously, run `mint update` to ensure parity with production rendering.
  </Step>

  <Step title="Start the preview server">
    ```bash
    mint dev
    ```
    Visit `http://localhost:3000` to confirm live reload works. CLI output calls out broken links and missing frontmatter automatically.
  </Step>
</Steps>

## Step 2 · Brand the experience

<AccordionGroup>
  <Accordion icon="palette" title="Update global identity">
    1. Open `docs.json` and edit `name`, `colors`, and `logo` fields.
    2. Save the file—your preview reflects changes immediately.
    3. Check light and dark themes to verify contrast ratios.
  </Accordion>

  <Accordion icon="map" title="Curate navigation">
    1. Add or reorder pages inside `navigation.tabs[].groups[].pages`.
    2. Use folder-prefixed slugs (for example `backend/scala3-contextual-abstractions`).
    3. Keep the sidebar under seven items per group for readability.
  </Accordion>

  <Accordion icon="wand-magic-sparkles" title="Wire up AI helpers">
    Bootstrap `.cursor/rules.md`, `.windsurf/rules.md`, or `CLAUDE.md` using the templates in the **AI tools** section so every collaborator gets the same guidance.
  </Accordion>
</AccordionGroup>

## Step 3 · Validate and deploy

<Steps>
  <Step title="Run automated checks">
    ```bash
    mint broken-links
    ```
    Fix any reported issues and rerun until you get a clean pass.
  </Step>

  <Step title="Commit with context">
    ```bash
    git add .
    git commit -m "docs: refresh quickstart + api reference"
    git push origin main
    ```
  </Step>

  <Step title="Deploy via GitHub app">
    Install the Mintlify GitHub app from your <a href="https://dashboard.mintlify.com/settings/organization/github-app">dashboard</a>. Once connected, every push to the default branch triggers a deployment and posts status checks back to GitHub.
  </Step>
</Steps>
