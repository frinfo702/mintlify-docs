## Failure modes

<ResponseField name="error" type="integer" required>
`404` indicates the plant does not exist, `401` indicates invalid credentials. Both return a JSON body with this schema even though the success case is empty.
</ResponseField>

<ResponseField name="message" type="string">
Additional diagnostic info.
</ResponseField>
