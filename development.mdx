<Info>
  <strong>Prerequisites</strong>
  <ul>
    <li>Node.js 18.17+ (LTS) and npm 9+</li>
    <li>Access to the repository that contains <code>docs.json</code></li>
    <li>Optional: Docker Desktop if you plan to run Dev Containers</li>
  </ul>
</Info>

## Install once

```bash
npm i -g mint
```

<Tip>Upgrade periodically with <code>mint update</code> to stay aligned with the renderer that powers production deployments.</Tip>

## Run the preview server

<Steps>
  <Step title="Launch">
    In the folder that contains `docs.json`, run:

    ```bash
    mint dev
    ```

    The CLI prints a localhost URL (default `http://localhost:3000`) and watches every MDX/JSON edit.
  </Step>

  <Step title="Customize the port">
    ```bash
    mint dev --port 3333
    ```

    If the specified port is busy, Mintlify automatically falls back to the next available one and logs the change.
  </Step>

  <Step title="Verify output">
    - Use the on-screen console to catch missing frontmatter and invalid components.
    - Run `mint broken-links` before every PR to surface orphaned references.
    - For API-heavy pages, add the `--schema api-reference/openapi.json` flag to lint endpoint definitions.
  </Step>
</Steps>

## Recommended editor setup

- **VS Code**: install the <a href="https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx">MDX</a> extension plus <a href="https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode">Prettier</a> for formatting.
- **Cursor/Windsurf**: load the project rules documented under **AI tools** so assistants follow our style guide.
- **Linting**: enable markdownlint for quick feedback on heading hierarchy and spacing.

## Troubleshooting playbook

<AccordionGroup>
  <Accordion title="Error: Could not load the 'sharp' module on Apple Silicon">
    1. Uninstall the current CLI: `npm remove -g mint`
    2. Upgrade Node to the latest LTS.
    3. Reinstall: `npm i -g mint`
  </Accordion>
  <Accordion title="Preview server fails silently">
    Delete the cache folder with `rm -rf ~/.mintlify`, then rerun `mint dev`.
  </Accordion>
  <Accordion title="Fonts or images missing in preview">
    Restart the server with `mint dev --clear-cache` to rebuild assets.
  </Accordion>
</AccordionGroup>
